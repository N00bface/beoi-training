# Unit 12: Greedy
This unit covers several greedy algorithms and their basic characteristics

- Coin change
- Interval scheduling
- Load balancing

Complementary notes can be found in section 3.4 of the book Competitive Programming 3.

## Prerequisites

- Unit 1: [Complexity](../01-complexity)
